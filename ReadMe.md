# UXStream Auditor

UXStream Auditor is an AI-powered tool that analyzes recorded videos of users interacting with a web, app, or game interface and generates a structured UX report highlighting usability issues, friction points, and improvement opportunities.

The goal is simple: identify where users struggle, why it happens, and how to fix it before those problems turn into churn.

## âš ï¸ Events

Events represent specific moments in the video where the system detected potential UX problems.

Each event includes:
- Timestamp
- Event type
- Description
- Severity level

### Event Types

Classification of user friction with the product:

- **Error**  
  Action failed to meet the intended goal.

- **Pause**  
  Unintended or unexplained stop in user activity.

- **Repetition**  
  Redundant actions caused by unclear feedback or interface confusion.

- **Hesitation**  
  Uncertainty or delay in decision-making.

### Severity Levels

Impact on overall User Experience:

- **High**  
  Prevents task completion or causes significant frustration.

- **Medium**  
  Noticeable break in user flow.

- **Low**  
  Minor annoyance with limited impact.

## ðŸ”® Scenarios

Scenarios simulate the potential outcome of applying each recommendation.

Each scenario includes two key indicators:

### Score

- A predictive metric from **0 to 100**
- Represents expected usability and user satisfaction after applying the change

### Risk

The likelihood of negative consequences:

- **High**  
  Significant change that may alienate users or require high development effort.

- **Medium**  
  Possible minor side effects.

- **Low**  
  Safe and low-impact improvement.
